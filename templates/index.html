Flask Blog Application

A multi-user blog platform with secure authentication and admin-only permissions. Built with one-to-many database relationships between users, posts, and comments, using WTForms for input handling and CKEditor for rich text. Deployed live with PostgreSQL on Render. User profile images are handled automatically through Flask-Gravatar, while session-based login ensures only authenticated users can comment or interact with posts. Configuration secrets like database URLs and API keys are kept secure using environment variables, following production-ready deployment practices

Movie Budget Analysis

An exploratory data analysis and linear regression project examining the relationship between film production budgets and worldwide box office revenue. Found that while the average film costs ~$31M and returns roughly 3x that, the regression revealed budget is actually a weak predictor of revenue for older films — many low-budget films significantly outperformed high-budget ones. For modern films, the model fits much better, predicting a $150M budget film would earn just under $500M. Visualized using Seaborn scatter and bubble charts, with regression modeling via scikit-learn.

Python Pandas Seaborn Scikit-learn Linear Regression

Flight Deal Finder

An automated tool that monitors flight prices from London to multiple destinations and sends real-time SMS alerts when a cheap fare is found. Pulls destination and price data from Google Sheets, resolves airport codes, searches live flight prices, and triggers Twilio SMS notifications — orchestrating four separate APIs into one automated pipeline. Sensitive credentials like API keys are stored securely using python-dotenv, keeping them out of source control. The script runs on a schedule, continuously checking for new deals without manual intervention, making price monitoring fully hands-off once configured.
